Answers

1. What does HTML stand for?

a) Hyper Text Markup Language


2. What is the PRIMARY function of HTML in website development?

c) Providing the foundation and structure

3. Which of the following is NOT a common way to use JavaScript on a website?

b) Defining the overall look and feel of the website


4. What is the main advantage of using JavaScript frameworks in web development?

b) They provide pre-written code and functionalities, saving time


5. What is the BEST analogy used in the article to describe the relationship between HTML, CSS, and JavaScript?

b) Bricks, paint, and electrical wiring


6. What is the primary purpose of a JavaScript framework in web development?

c) To simplify complex web application development


7. What is the primary purpose of the < head > section in an HTML document?

b) To contain metadata about the document, not visible on the page itself.


8. What is the difference between a LINK tag and an SCRIPT tag in HTML?

a) Both are used for including external resources, but link is for stylesheets and script for JavaScript files


9. When responding to user input on a web page, what security concern should be considered when using JavaScript?

c) Validating user input to prevent malicious code injection


10. What is a common use case for local storage in web development?

a) Storing user preferences (like language settings) between browsing sessions
